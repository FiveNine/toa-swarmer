# Swarmer
This plugin displays numbers on scarab swarms in the kephri room as they spawn.

![](https://imgur.com/msneEOQ.png)

## Third Party Compliance
This plugin does not indicate `Indicates the time where a boss mechanic may start or end`.
This plugin's feature is similar to the `Nyloer` plugin which counts the waves in the nylo room.
